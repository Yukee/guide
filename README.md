# DMT guide
* The goal of this project is to write a health and safety website directed at cluster headache patients who use DMT to abort their attacks
* The website should be comprehensive, covering how to safely abort with smoked DMT, medical considerations of taking DMT, etc.
* And the same time, the website should be accessible. The median reader is a 50-something with no experience with drugs, and little skill in finding reliable information online.
# Roadmap
* The current document is pretty daunting to read for our median reader. It could for instance use more illustrations or photos. This is in particular true of the "Aborting with DMT" part.
* The markdown document will eventually be turned into a website.
  Claude and the user will iterate on markdown document together, until it reaches a satisfactory state.
* External feedback will be received, and the last step will be repeated until a stable version is reached
* The website will be constructed, and the feedback phase will be repeated until convergence on the website
